What is Una?

Una is a free and open source Javascript library that allows developers to create interactive applications where multiple users can interact with large central displays in real-time using mobile devices.

Why use Una?

Cross-display Interaction

Una makes it easy for you to create interactive cross-display applications where users can interact with large displays using their mobile devices.

Built for the Web

Entirely web-based, no need to download any application from the app store; your web browser is the only app you need. No special hardware needed.

Easy Integration

Una handles the server-side for you, you only need to work on client-side code. This frees you up from working with low-level transport mechanism so you can focus on making your kick-ass applications.
